PML_project_apalleja

This repository contains the R markdown document for the final project to complete the Practical Machine Learning course of Coursera. We try to predict how an exercise was done (classes) based on 159 predictors.

The training data for this project are available here:

https://d396qusza40orc.cloudfront.net/predmachlearn/pml-training.csv

The test data are available here:

https://d396qusza40orc.cloudfront.net/predmachlearn/pml-testing.csv
